Je hebt twee vaak gebruikte kleursystemen: RGB en CMY. RGB is het kleursysteem dat we gebruiken om meer licht toe te voegen aan iets zwarts, zoals beeldschermen. Het heet een additief kleursysteem. Je hebt de kleuren rood, groen en blauw. Deze kleuren mix je om alle andere kleuren te maken. Als je het maximum van alle kleuren gebruikt, krijg je wit. Voor alle drie de kleuren wordt een aparte byte gebruikt. Een byte heeft een waarde van 0 tot 255. We gebruiken voor RGB dus 3 bytes. Voor elke byte gebruiken we 2 tekens om een code ervan te maken. Het gebruikt 0123456789ABCDEF. Dit is hexadecimaal. Een code voor RGB waardes ziet er als volgt er uit: #34A2F7. Dat is een soort licht blauw. De eerste 2 cijfers staan voor rood, de middelste 2 voor groen en de laatste 2 voor blauw.

Als je wilt oefenen met zelf rgb waardes lezen kan je dat doen op deze website

CMY-K is een kleure systeem met de kleuren cyaan, magenta, geel en zwart. het is een subtractief kleurensysteem. Dit systeem gebruik je om licht weg te halen van iets wit, zoals papier. Je gebruikt deze kleuren om alle andere kleuren te maken. Als je maximum van alle kleuren doet krijg je zwart. Het gebruikt ook 3 bites, maar de K (zwart) is een extra bite die gebruikt wordt om zwart nog zwarter te maken. Dit komt omdat als je alle kleuren op maximum zet krijg je een soort donker bruin. Deze bite wordt ook gebruikt om inkt te besparen. CMY-K wordt vaak gebruikt bij printers.
